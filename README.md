# Hackintosh EFI Config

## Setup

- mother board: Asus ROG B460i
- CPU: Intel i9 10900
- RAM: 2x16GB DDR4 2933Mhz
- GPU: Asus Tuff 6500XT
- SSD: WD Black 770
- Wireless & BT: Intel Wireless AX200

## System

- booter: OpenCore
- OS: macOS Monterey 12.0+

## BIOS Tunning

Disable:

- Fast boot
- VT-d
- CSM
- Intel SGX

Enable:

- VT-x
- Hyper Threading
- OS type: other system
